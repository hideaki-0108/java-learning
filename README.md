# java-learning

Javaを学習するためのリポジトリです。バージョンやFWに縛りはつけない予定です。

## 学習目的

フロントエンドエンジニアとして3年目になります。これまでTypeScript、React、Vueを使ってきましたが、そろそろバックエンド、特に自作のAPIが作れるようになりたいと思うようになりました。

業務ではSpring Bootを利用しているので、Javaの基礎から学んでいこうと考えています。現場でもJavaが使われているため、学習を進めていきます。

## 学習計画

まずはJavaの基本を学びます。基本的な文法やオブジェクト指向プログラミング、コレクション、例外処理、ストリームAPIなどを学習する予定です。

次に、AtCoderの問題をマイペースに解いていこうかと考えています。アルゴリズムやデータ構造を実践的に学びながら、効率的なコードの書き方を身につけたいです。

その後は、Spring Bootを使ったAPI開発に取り組みます。データベース連携や認証・認可など、実務で必要になる機能を実装していく予定です。

## 技術スタック

- Java
- Spring Boot（業務で使用中）

その他、必要に応じて随時追加していきます。

## 今後の展望

フロントエンドとバックエンドの両方の知識を持つフルスタックエンジニアとして成長し、今後のキャリアに活かしていきたいです。
